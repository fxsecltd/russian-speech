Помощь

Это приложение для развития навыков русской речи с помощью Google Speech Recognition API:

  • Нахмите кнопку "Запись" и держите нажатой до окончания записи фразы.
  • Голос будет распознан через Web Speech API и сформирована картина несоответствий слов.
  • Прослушайте свою речь нажав на кнопку "Слушать" и исправьте дефекты!.

Фразы сгруппированы в текст: одна строка - одна фраза.
Вы можете выбрать текст из списка или создать свой собственный, щеокнув в левом верхнем уголке.

Keep in mind
Распознавание речи не всегда совершенно и может ошибочно распознавать речь.
Некоторые фразы могут быть распознаны некорректно даже при идеальном произношении.

Рекомендуется
Используйте высококачественные микрофоны для улучшения качества распознавания.
